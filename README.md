Mục tiêu:

Giám sát môi trường nông trại (nhiệt độ, độ ẩm, ánh sáng, nồng độ CO₂, độ ẩm đất, mưa).

Điều khiển tự động thiết bị: bơm nước, quạt làm mát, mái che servo, LED chiếu sáng.

Hỗ trợ điều khiển từ xa qua MQTT và hiển thị dữ liệu trực tiếp trên LCD 16x2.

Vấn đề giải quyết:

Tăng hiệu quả canh tác nhờ tự động hóa.

Giảm công lao động và hạn chế thất thoát nước, năng lượng.
